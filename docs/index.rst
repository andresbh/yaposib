Welcome to yaposib's documentation!
===================================

Yaposib is a python binding to OSI, the Open Solver Interface from
COIN-OR. It intends to give access to various solvers through python.
Yaposib was created to be integrated with pulp-or, and plays nicely with
it.

Various Infos
=============

The repository is hosted on github_.

::

    git clone https://github.com/coin-or/yaposib.git

If you are unable to use git, github offers nice features, such as
checkouting the code from svn.

::

    svn checkout https://github.com/coin-or/yaposib
